# Email Survey App
Web App that sends out surveys via email and takes payments via Stripe.
Allows users to create customizable email survey campaigns and record feedback to the surveys
for internal use.
Built with MERN stack using SendGrid API, Stripe API, NGROK, MaterializeCSS, PassportJS, Zod, and Cypress.
